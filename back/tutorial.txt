
-criar env
py -m venv .venv

-ativar env
 ./venv/Scripts/activate

-instalar depedências
pip install -r requirements.txt

-fazer migrações
py manage.py makemigrations

-aplicar migrações
py manage.py migrate

-rodar o servidor
py manage.py runserver

(Não precisa se ja estiver com o banco no back)
-puxar cidades e paises para o banco de dados, nas configs eu filtrei somente por paises do brasil
python manage.py cities_light

Você precisa cadastrar tudo, por que não subimos o nosso banco
